# MPC-to-streamerbot
A python script to send data to MPC-HC, specifically designed to use streamer.bot as an intermediary through a text file

# SETUP
1. download the script and make sure you have python installed
2. run the file inh python once to make sure the text file has been generated
3. in streamer.bot import the code provided
4. adjust the "write to file" action so it targets the queue file
5. adjust the "run a program" action so it targets your version of pythonw(version that doesn't show a command line lol)
6. try the command to verify it works
